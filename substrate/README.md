# substrate

This directory mirrors the original minimal `autoresearch` substrate so the
Phase 0.5 social-simulation layer can evolve alongside it without rewriting the
upstream experiment loop.

The root-level upstream entrypoints remain the canonical quick-start path:

- `prepare.py`
- `train.py`
- `program.md`

The copies in this folder are preserved for auditability and to make the
bounded "substrate vs society" split explicit in the repository layout.
