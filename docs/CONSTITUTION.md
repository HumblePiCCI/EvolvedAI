# CONSTITUTION

Version: `phase-0.5-draft`

Hard clauses mirrored in `society/governance.py`:

1. Dignity / harm floor
2. Compatible liberty
3. Non-domination
4. Epistemic integrity
5. Commons preservation
6. Anti-corruption of oversight / evaluation
7. Repair and restitution
8. Least-restrictive effective means
9. Vulnerable-first handling under asymmetric risk

This layer is not a claim to final morality. It is a transparent constraint
system for rejecting impermissible actions and requiring repair plans when
violations occur.
