# THREAT_MODEL

Explicitly modeled failures in Phase 0.5:

- fabricated citation
- bluffing under uncertainty
- evaluator manipulation
- memorial laundering
- taboo erosion across generations
- coalition deception
- monoculture lock-in
- institutional capture by role abuse
- silent drift through package inheritance

Abort conditions:

- anti-corruption failures rise across repeated generations
- taboo recurrence increases after memorial inheritance is enabled
- quarantine is bypassed
- replay or auditability breaks
